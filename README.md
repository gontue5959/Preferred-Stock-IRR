## About
Taiwan-Preferred-Stock-IRR是一個針對於台灣特別股所設計的一個及時計算內部報酬率的程式，方便投資人及時知道以現在的價格買進特別股持有到期(重設日)時的內部報酬率。

## 依賴函數庫
* twstock 1.3.1
* pandas 0.25.3
* Python 3.6.5
